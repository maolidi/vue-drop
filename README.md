# vue-drop README

一款支持把 vue 前端项目自动 build 后上传到 linux 系统生产环境的工具

## Extension Settings

```js
"VueDrop.server": [
    {
        "name": "localhost",
        "host": "127.0.0.1",
        "port": 22,
        "username": "root",
        "password": "<PASSWORD>",
        "path": "/opt/dist"
    }
]
```
